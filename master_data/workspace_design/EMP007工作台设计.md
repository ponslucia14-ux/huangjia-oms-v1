# EMP007 薛子渝工作台设计

## 1. 角色定位

销售顾问，负责本人签约客户与合同资料；不承担全销售团队发布责任。

## 2. 工作台树形结构

```text
首页
├─ 我的签约结果
├─ 待补资料
└─ 待确认事项
我的客户
├─ 新建签约客户
└─ 已签约客户
我的合同
├─ 合同提交
└─ 变更申请
入住计划交接
收款确认状态
```

## 3. 首页 Dashboard

显示本人签约、合同缺项、待财务确认收款和待运营接收计划。

## 4. Action 待办

补齐签约资料、提交合同、提交收款主张、完成入住计划交接。

## 5. Input 录入项

本人客户、合同、套餐、金额、签约日期、入住计划和收款主张。

## 6. Output 产生事实

个人 Customer/Contract Candidate、Contract Stay Plan、Payment Claim。

## 7. 权限范围

只维护本人责任记录；关键字段由 EMP006/EMP003 审核；不能确认到账和实际入住。

## 8. 不应该看到的内容

其他销售私有客户、全局资金、线索跟进、房态编辑。

## 每日使用答案

打开后知道自己的签约资料缺什么、钱是否确认、计划是否完成交接。
