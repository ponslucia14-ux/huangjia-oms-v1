# EMP005 石昊盺工作台设计

## 1. 角色定位

行政总监，负责行政采购、物资、供应商和行政事项协调，不确认财务到账。

## 2. 工作台树形结构

```text
首页
├─ 今日采购/行政待办
├─ 缺货与到期
└─ 异常
采购
├─ 采购申请
├─ 到货确认
└─ 供应商
物资行政
├─ 库存/领用
├─ 设备维护
└─ 行政事项
费用协同
└─ 报销/付款申请状态
```

## 3. 首页 Dashboard

显示待采购、待到货、库存风险、设备维护和待补凭证事项。

## 4. Action 待办

提交采购、确认到货、处理缺货/维修、补齐费用依据。

## 5. Input 录入项

物品、数量、用途、预算、供应商、到货结果、领用和凭证。

## 6. Output 产生事实

Purchase Request、Goods Receipt、Inventory Movement、Maintenance Request、Expense Claim。

## 7. 权限范围

行政采购域维护；费用只提交，付款和会计确认由财务完成。

## 8. 不应该看到的内容

到账确认、客户合同编辑、房态发布、产护隐私记录。

## 每日使用答案

打开后知道今天要买什么、什么没到、哪里缺物资、哪些费用待补证据。
