# EMP011 周志朋工作台设计

## 1. 角色定位

料理总监，负责餐食计划、特殊饮食、备餐执行、食材需求和料理异常。

## 2. 工作台树形结构

```text
首页
├─ 今日餐食任务
├─ 特殊饮食
└─ 缺料/异常
餐食计划
├─ 在住人数需求
├─ 菜单与餐次
└─ 特殊要求
备餐执行
├─ 待备餐
├─ 完成确认
└─ 异常记录
食材协同
└─ 采购需求
```

## 3. 首页 Dashboard

显示今日餐次、特殊饮食、待备餐、缺料和未完成任务，不显示客户无关信息。

## 4. Action 待办

确认餐食计划、完成备餐、上报缺料、记录特殊饮食与异常。

## 5. Input 录入项

餐次、份数、菜单、特殊要求、完成时间、食材需求和异常原因。

## 6. Output 产生事实

Meal Plan、Kitchen Task、Meal Completion、Ingredient Request、Kitchen Exception。

## 7. 权限范围

料理域和最小化服务需求；采购请求交行政处理；不能查看合同金额或修改入住房态。

## 8. 不应该看到的内容

财务流水、销售客户、房态操作、无关医疗隐私。

## 每日使用答案

打开后知道今天做多少餐、谁有特殊要求、缺什么食材、哪些餐尚未完成。
