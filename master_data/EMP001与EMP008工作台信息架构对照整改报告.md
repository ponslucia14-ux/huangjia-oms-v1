## 一、岗位MD定义摘要

### EMP001 石磊 / 主理办

- 首页看什么：经营总览、风险、审批、资金、在住与房态。
- 主要动作：审批例外、经营决策、查看异常。
- 主要输出：审批结果、经营指令。
- 岗位边界：不替代销售、财务、运营岗位完成日常录入。

工作主线：看全局 → 看异常 → 看待决策事项 → 审批或授权 → 形成经营指令 → 追溯执行结果。

### EMP008 刘芳羽 / 店总兼销售

- 首页看什么：房态、入住、出馆、异常和本人销售待办。
- 主要动作：确认真实入住与出馆、发布房态、排房调度、处理运营异常、完成本人销售工作。
- 主要输出：当前入住、当前房态、排房与运营处理结果、本人销售事实。
- 岗位边界：不确认财务到账、不替代管家录入全部在住资料、不处理其他销售人员客户。

工作主线：看今天运营 → 确认入住与出馆事实 → 调度未来房间 → 处理房态和异常 → 完成本人销售工作 → 留下运营结果。
